* This is a WIP for an multiplayer isometric game codenamed Bloodshed.                 *
*                                                                                      *
* It is written in C++ using the SDL library.                                          *
*                                                                                      *
* Current developers:                                                                  *
*	- Neagu Cristian Doru (Fireeyes)                                               *
*	- Popa George (QSilver)                                                        *
*                                                                                      *
* Big TO-DO's                                                                          *
*                                                                                      *
* - Map-editor                                                                         *
* - Particle System using CEntity (each particle being a CEntity)                      *
* - Define character (it has no own properties)                                        *
* - UI                                                                                 *
* - Collision between two CEntities                                                    *
* - Text Engine (something like drawText(char* Text, int X, int Y, color, font)        *
* - Collision between CEntities and world                                              *
*                                                                                      *
* Small TO-DO's                                                                        *
*                                                                                      *
* - Check on which tile is the mouse over                                              *
* - Get the tile that has been clicked                                                 *
* - Get the tile that an CEntity is on                                                 *
* - Fix player animation                                                               *
* - Make the player walk on 8 directions rather than 4                                 *
